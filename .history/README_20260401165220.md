# aro-lummiData-website

GitHub Pages deployment is configured with GitHub Actions.

## What is configured

- Workflow: `.github/workflows/deploy.yml`
- Published artifact: `dist/`
- Entry file mapping: `code.html` -> `index.html`
- Custom domain file: `CNAME` (`lumii-data.io`)

## One-time GitHub setup

1. Push this repository to GitHub.
2. Open repository `Settings` -> `Pages`.
3. Under `Build and deployment`, set `Source` to `GitHub Actions`.

## DNS setup for `lumii-data.io`

In your DNS provider, add these records:

- `A` record for root (`@`) -> `185.199.108.153`
- `A` record for root (`@`) -> `185.199.109.153`
- `A` record for root (`@`) -> `185.199.110.153`
- `A` record for root (`@`) -> `185.199.111.153`
- `CNAME` record for `www` -> `<your-github-username>.github.io`

Then in GitHub `Settings` -> `Pages`, set custom domain to `lumii-data.io` and enable HTTPS after DNS is propagated.
